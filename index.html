Cuculi AI Recommendation Engine

Personalized event recommendations and psychology-driven nudges designed to re-engage dormant users with precise targeting and compelling content.

Overview

Cuculi AI is a microservice ecosystem purpose-built for personalized event recommendations and behavioral nudges. The platform combines user intelligence, similarity scoring, and message generation to convert dormant users through tailored outreach.

Key Features

Personalized Recommendations

Match users to relevant events using advanced vector similarity algorithms that adapt to their evolving interests.

Smart Targeting

Score user compatibility to prioritize outreach and deliver the right nudges to the right audience at the right time.

AI Message Generation

Generate psychology-driven messaging that resonates with dormant users and inspires renewed engagement.

Architecture

MongoDB Vector Search

High-performance vector search powers event-to-user matching with contextual awareness and semantic understanding.

OpenAI APIs

Integrate generative AI capabilities to produce nuanced nudges and personalized content at scale.

API Microservices

Modular services handle recommendations, targeting, and messaging for reliable and maintainable operations.

API Endpoints

Event Recommendations

POST /api/users/{userId}/recommendations — Returns curated event suggestions for each user.

User Targeting

POST /api/events/{eventId}/targeting — Identifies high-fit audiences for event promotion.

Message Generation

POST /api/messages/generate — Crafts AI-generated nudges optimized for conversion.

Deployment

AWS Lambda

Deploy the microservices serverlessly with serverless deploy for scalable, managed infrastructure.

Flask Local

Run locally during development using python app.py to iterate quickly on features and integrations.
